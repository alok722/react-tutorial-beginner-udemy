React.createElement accepts 3 arguments minimum,
React.createElement('div',null,'h1','My name is Alok'); // Output: 
<div>"h1" My name is Alok</div>

// h1 is not taken as tag
To do so,:
React.createElement('div',null,React.createElement('h1',{className:'App'},'My name is Alok'));
// <div><h1>My name is Alok</h1></div> with style
// We dont explicitely use like this, bcoz complilation take care of this.

In general scenario, only one parent element should be returned.
